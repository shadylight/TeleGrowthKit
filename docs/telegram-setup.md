# Telegram Setup

1. Create bot with BotFather and capture token.
2. Set bot username in `BOT_USERNAME`.
3. If using channel-gate, set `REQUIRED_CHANNEL_ID` and optional `REQUIRED_CHANNEL_HANDLE`.
4. Add bot as channel admin if membership checks are needed.
5. Set `ADMIN_TELEGRAM_IDS` as comma-separated numeric user IDs.
6. Set webhook with optional `TELEGRAM_SECRET_TOKEN`.
